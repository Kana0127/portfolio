# Monthly Step

## 1. サービス概要

### このサービスはどんなサービスか

このサービスは、忙しい社会人が月ごとの目標を週単位の小さな行動に分解し、日々の取り組みを簡単に記録できる目標管理アプリです。

年始や月初に立てた目標が日々の忙しさに埋もれてしまう人に向けて、「今週何に取り組むか」を見える化します。

日々の小さな前進を記録し、週次・月次で振り返ることで、目標に向かって進んでいる実感と自己肯定感につなげます。

### 誰の、どんな課題を、どう解決するか

仕事・家事・育児などで忙しく、自分のやりたいことが後回しになりがちな社会人に向けたサービスです。

年始や月初に目標を立てても、日々の忙しさで忘れてしまったり、目標が大きすぎて何から始めればよいか分からず行動に移せなかったりする課題を解決します。

月単位で立てた目標を週単位の小さな目標に分解し、日々の取り組みを簡単に記録・可視化することで、目標に向かって前進している実感を得られるようにします。

## 2. このアイデアはどこから生まれたか

自分自身が昔から、年始や月初に目標を立てても、日々の忙しさに追われるうちにその目標を忘れてしまったり、目標が大きすぎて何から始めればよいか分からず行動に移せなかったりする経験がありました。

その結果、立てたはずの目標がいつの間にか頭から消え、気づけば日々だけが過ぎてしまっていることに歯がゆさを感じていました。

そこで、目標を月・週単位の小さなステップに分解し、「今」何に取り組めばよいのかを見える化できれば、忙しい毎日の中でも行動に移しやすくなるのではないかと考えました。

### 2-1. きっかけとなった体験・感情

年の初めに「今年こそは英検1級に挑戦するぞ」「行きたい場所に行くぞ」という目標を立ててみるものの、1年というスパンが長すぎることや、日々の忙しさに追われることで、目標を立てたこと自体を忘れてしまうことがありました。

意欲はあるにもかかわらず、それらの目標を管理できていないことで、せっかくの自分の時間を無駄にしてしまっているように感じ、歯がゆい思いをしていました。

### 2-2. なぜそれが気になったのか

学生時代から、テストの点数や貯金額など、自分の努力が成果として形に現れることで「自分は以前よりも前に進んでいる」「成長できている」と実感できていました。

一方で、社会人になってからは、大半の時間が仕事で過ぎていく中で、目に見えて前進できている証が少なくなり、焦燥感を感じるようになりました。

## 3. 課題の整理

### 3-1. 表に見えている困りごと

- ノートに「1年でやりたいことリスト」を作成しても、そのノート自体を持ち歩く頻度が減ると、目標を立てたことを忘れてしまう
- Todoアプリなどに1年の目標を書いても、その目標が大きすぎることで「いつまでに何からすべきなのか」が分からなくなり、後回しにしてしまう
- 目標を書いても、その目標が先のことすぎて現実味が湧かず、意欲が薄れてしまう

### 3-2. 本当に解決したい課題は何か

表面的な困りごとは、目標を立てても日々の忙しさで忘れてしまうことです。

#### なぜ？

目標が大きすぎて、今日・今週何をすればよいか分からないからです。

#### なぜ？

具体的な行動に落とし込めていないと、行動に移すハードルが高くなり、後回しにしてしまうからです。

#### なぜ？

後回しが続くと、「自分は何も前に進めていない」と感じ、自己肯定感が下がってしまうからです。

### 本当に解決したい課題

やりたいことはあるのに日々のルーティンに埋もれてしまい、前に進めていないような停滞感を感じ、自己肯定感が下がってしまうこと。

## 4. 想定ユーザーについて

### 4-1. 想定しているユーザー

| 項目 | 内容 |
|---|---|
| 年齢 | 仕事や家事、育児に忙しい20代後半〜30代後半の社会人 |
| 生活・状況 | 家事・育児・仕事など、自分自身の役割や責任がある中で、自分の挑戦したいことに時間を使いにくい |
| 抱えている課題 | 挑戦したいことはあるものの、自分の時間が限られているため、何から始めればよいか分からず後回しにしてしまう |
| アプリで解消したいこと | 月ごとの目標を週単位の小さな目標に分解し、日々の取り組みを簡単に記録できるようにすることで、忙しい中でも前進を実感できるようにする |

### 4-2. 自分とユーザーの距離

最も近いものは、自分自身・過去の自分です。

私自身、年始に「英検1級に挑戦したい」「行きたい場所に行きたい」といった目標を立てても、日々の仕事や生活に追われる中で、いつの間にか目標を忘れてしまう経験がありました。

一方で、学生時代からテストの点数や貯金額など、自分の努力が目に見える形になることで前に進んでいる実感を得ていました。

そのため、忙しい日々の中でも小さな前進を見える化できれば、同じように目標を後回しにしてしまう人の助けになると考えました。

### 4-3. 実際に使われる可能性について

#### 最初に使ってくれそうな人

資格勉強、運動、読書、旅行準備など、やりたいことはあるものの、仕事や家事で後回しにしがちな20代後半〜30代後半の社会人。

#### このアプリを思い出す場面

月初に目標を立てたものの、数週間後に「結局何も進んでいない」と感じたとき。

また、Todoアプリにタスクを書いても、大きな目標とのつながりが見えず、続かなかったとき。

#### 今の生活で代わりに使っているもの

- 紙のノート
- Todoアプリ
- カレンダーアプリ
- Notion
- スマホのメモ

#### 置き換えてまで使う理由

一般的なTodoアプリはタスク管理が中心ですが、このアプリでは月目標を週単位に分解し、日々の取り組みを簡単に記録できます。

そのため、忙しい日でも「今週何に取り組むか」が分かりやすく、さらに小さな前進を可視化することで自己肯定感につなげられます。

### 4-4. ユーザーがサービスを導入して利用するまでのイメージ

#### どんな場面・困りごとがきっかけで、このサービスを知る・思い出すか

数年前からずっと「英検1級合格」と「東京マラソン」に挑戦してみたいと思っていた30代前半のAさん。

しかし、毎日の仕事や家事などで日々の時間が過ぎていき、ただ行動に移せず時間だけが過ぎてしまっている。

#### 最初の一歩でユーザーがすること

- アカウントを作成する
- 今月の目標を数個作成する
- 必要に応じて、各週に何をするのかを計画する

#### 利用を続ける理由

- 月目標を週単位の小さな目標に分解することで、アプリを開いたときに「今週何に取り組むか」が分かる
- 日々の記録は「◎：しっかりできた」「○：少しできた」「×：できなかった」の3択と任意コメントで簡単に残せる
- 週末には、週目標に対する体感達成率とコメントを記録し、次の週に向けて取り組み方を調整できる

## 7. このアプリで実現すること

### MVPで作る機能

| 機能名 | 内容 |
|---|---|
| ユーザー登録機能 | ユーザーがアカウントを作成できる |
| ログイン・ログアウト機能 | 登録したユーザーがログイン・ログアウトできる |
| 月目標作成機能 | 今月達成したい目標を登録できる。カテゴリーを設定できる。目標に応じて、週ごとの小さな目標に分解するか、単発目標として達成・未達成で管理するかを選べる |
| 週目標作成機能 | ステップ型の月目標に対して、週ごとの小さな目標を手動で登録・編集できる |
| マイページ機能 | 今月の目標、今週の目標、今日の記録状況を確認できる |
| 日次記録機能 | 週目標に対して、その日にどれくらい取り組めたかを「◎・○・×」で簡単に記録できる。任意で一言メモを残せる |
| 週次・月次振り返り機能 | 週・月の終わりに、目標に対する体感達成率とコメントを記録できる |

カテゴリー例：趣味、学習、遊び、スポーツ、美容・健康、読書、お金、その他

#### 画面イメージ

```text
今月の目標
英検1級の単語帳を1周する

今週の目標
1〜300語を覚える

今日の記録
今日、どれくらい取り組めましたか？

[ ◎ しっかりできた ]
[ ○ 少しできた ]
[ × できなかった ]

メモ
[ 通勤中に単語を20語確認できた ]

[ 記録する ]
```

### 本リリースで追加予定の機能

| 機能名 | 内容 |
|---|---|
| 月次レポート機能 | 日次記録や週次・月次振り返りをもとに、1か月の達成状況を自動で可視化する |
| 来月目標作成サポート機能 | 今月の振り返り内容をもとに、来月の目標作成へスムーズにつなげる |
| カレンダービュー機能 | 日次記録の達成度をカレンダー上にスタンプ形式で表示する |

| 年次レポート機能 | 1年間の目標達成状況や振り返りを確認できる |

| 達成記録の共有機能 | 任意で日々の達成記録を共有し、いいねやコメントで励まし合える |

## 8. このアプリの懸念点とその対策

### 懸念点①

| 項目 | 内容 |
|---|---|
| 何が問題になりそうか | 忙しい社会人を対象にしているため、目標の登録や日々の記録に手間がかかると、継続して使ってもらえない可能性がある |
| なぜそう思うか | 想定ユーザーは、仕事・家事・育児などで自分の時間が限られている人である。そのため、毎日細かく目標を設定したり、詳細な進捗を入力したりする必要があると、かえって負担になってしまうと考えた |
| そのために考えている対策 | 日ごとの目標を事前に細かく設定するのではなく、週目標に対してその日にどれくらい取り組めたかを「◎・○・×」で簡単に記録できる形にする。また、コメントは任意入力とし、忙しい日でも短時間で記録できるようにする |

## 9. 今後の展開・発展の方向性

### 今考えている発展案

#### 機能追加の方向性

MVPでは、月目標を週目標に分解し、日々の達成度を「◎・○・×」で記録するところまでを実装します。

今後は、日次記録や週次振り返りのデータをもとに、月次レポート・年次レポートを表示できるようにしたいです。

例えば、1か月のうち「◎・○・×」がどれくらいあったか、どのカテゴリーの目標に多く取り組めたか、週ごとの体感達成率がどのように変化したかを可視化することで、ユーザーが自分の行動傾向や小さな前進を振り返れるようにします。

また、将来的には年目標と月目標の紐づけ、リマインド機能、他ユーザーとの共有機能、AIによる週目標案の生成機能も検討します。

#### UI・体験の改善

忙しい人でも続けやすいように、日次記録はできるだけ少ない操作で完了できるUIにします。

アプリを開いたときに、今月の目標・今週の目標・今日の記録状況がすぐ分かる画面を目指します。

また、カレンダービューで日々の達成度をスタンプのように表示し、これまでの取り組みが視覚的に見えるようにします。

未達成の日があっても自己否定につながらないように、「少しできたこと」や「続けようとしたこと」も前向きに残せる体験にしたいです。

#### 機能以外の工夫

目標を立てて終わりにしないために、週末や月末に振り返りを行う導線を用意します。

振り返りでは、できなかった理由を責めるのではなく、「次の週・次の月はどう調整するか」を考えられるようにします。

また、将来的に共有機能を追加する場合は、ランキングのように他人と競争する仕組みではなく、任意共有・いいね・コメントによるゆるい応援を中心にします。

これにより、ユーザーが自分のペースで続けながら、小さな前進を自信に変えられるサービスを目指します。

## 10. 技術スタック

### 10-1. 使用予定の技術

| 項目 | 内容 |
|---|---|
| フレームワーク | Ruby on Rails 8.1.3 |
| DB | PostgreSQL |
| デプロイ先 | Render |

### 使用予定ライブラリ

| ライブラリ | 用途 |
|---|---|
| Sorcery | ログイン機能 |
| simple_calendar | 簡易カレンダー表示 |
| enum | カテゴリーや日次記録ステータスの管理 |
| Chartkick | 月次レポートなどのグラフ表示。本リリース以降で使用予定 |

### 10-2. 技術選定の理由

Ruby on Railsは、ユーザー登録、ログイン、目標作成、日次記録、振り返りなど、CRUD処理を中心としたWebアプリ開発に適しているため使用します。

このアプリでは、月目標・週目標・日次記録・週次振り返り・月次振り返りなど、複数のデータを関連付けて管理する必要があります。そのため、ActiveRecordのアソシエーションを活用し、ユーザーごとの目標や記録を整理して扱えるようにしたいと考えています。

また、日々の記録をカレンダー上で確認したり、将来的に月次レポートとして可視化したりすることで、ユーザーが自分の小さな前進を振り返れる体験を作りたいと考えています。

### 10-3. キャッチアップ不足の懸念

#### 1. 複数モデルの関連付け

月目標、週目標、日次記録、週次振り返り、月次振り返りなど、複数のモデルが関連するため、設計を誤るとデータの取得や表示が複雑になる可能性があります。

対策として、MVPでは「月目標 → 週目標 → 日次記録」を中心としたシンプルな構造に絞り、必要に応じて週次振り返り・月次振り返りを紐づける形で実装します。

#### 2. 機能過多による開発遅延

月次レポート、AI連携、共有機能、リマインド機能などを最初から作ろうとすると、MVPの完成が難しくなる可能性があります。

対策として、MVPでは「月目標作成」「週目標作成」「日次記録」「簡単な振り返り」に絞り、AI連携や共有機能などは本リリース以降の機能として整理します。
